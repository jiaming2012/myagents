```python
def fetch_mercury_balances():
    """Fetch balances from Mercury API for both personal and business."""
    balances = {}
    for key_name, env_var in [("personal", "MERCURY_PERSONAL_API_KEY"), ("business", "MERCURY_BUSINESS_API_KEY")]:
        api_key = os.environ.get(env_var)
        if not api_key:
            continue
        try:
            resp = requests.get(
                "https://api.mercury.com/api/v1/accounts",
                headers={"Authorization": f"Bearer {api_key}", "accept": "application/json"},
                timeout=15,
            )
            if resp.status_code == 200:
                for acct in resp.json().get("accounts", []):
                    if acct.get("status") == "active":
                        balances[acct["id"]] = {
                            "balance": acct.get("currentBalance", 0),
                            "name": acct.get("nickname") or acct.get("name", ""),
                            "key": key_name,
                        }
        except requests.RequestException:
            print(f"Warning: failed to fetch Mercury {key_name} balances", file=sys.stderr)
    return balances
```

```python
TOKEN_FILE = Path(__file__).parent / ".xero_token.json"
TOKEN_URL = "https://identity.xero.com/connect/token"

def load_token():
    if TOKEN_FILE.exists():
        return json.loads(TOKEN_FILE.read_text())
    return None

def save_token(token):
    TOKEN_FILE.write_text(json.dumps(token))

def refresh_token(token):
    response = requests.post(
        TOKEN_URL,
        data={"grant_type": "refresh_token", "refresh_token": token["refresh_token"]},
        auth=(CLIENT_ID, CLIENT_SECRET),
        headers={"Content-Type": "application/x-www-form-urlencoded"},
    )
    response.raise_for_status()
    new_token = response.json()
    save_token(new_token)
    return new_token

# SDK init:
api_client = ApiClient(
    Configuration(oauth2_token=OAuth2Token(client_id=CLIENT_ID, client_secret=CLIENT_SECRET))
)
api_client.set_oauth2_token(token)
identity_api = IdentityApi(api_client)
connections = identity_api.get_connections()
tenant_id = connections[0].tenant_id
accounting_api = AccountingApi(api_client)
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI -> Xero API | OAuth2 tokens cross network boundary; refresh tokens are long-lived credentials |
| .xero_token.json -> filesystem | Token file contains OAuth2 access + refresh tokens on disk |
| .env -> process | XERO_CLIENT_ID and XERO_CLIENT_SECRET loaded from environment |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01.1-01 | Information Disclosure | .xero_token.json | mitigate | File added to .gitignore; contains OAuth2 tokens that must not be committed. Verify with `grep -q '.xero_token.json' .gitignore` |
| T-01.1-02 | Information Disclosure | .env (XERO_CLIENT_ID/SECRET) | mitigate | .env already in .gitignore (line 1). Client credentials loaded via os.getenv, never hardcoded |
| T-01.1-03 | Spoofing | TOKEN_URL endpoint | accept | Using hardcoded `https://identity.xero.com/connect/token` -- official Xero identity endpoint. TLS provides server authentication |
| T-01.1-04 | Tampering | .xero_token.json | accept | Single-user CLI tool on developer machine. File permissions are user-default. No multi-user risk |
| T-01.1-05 | Denial of Service | Xero API rate limits | mitigate | Graceful degradation: on any API error, warn to stderr and return empty dict. CLI continues with Monarch data only |
| T-01.1-06 | Information Disclosure | Token in refresh_token HTTP body | accept | POST body over HTTPS to identity.xero.com. Standard OAuth2 refresh flow -- no logging of request body |
</threat_model>
