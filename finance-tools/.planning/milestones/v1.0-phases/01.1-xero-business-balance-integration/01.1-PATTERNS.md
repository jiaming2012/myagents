**Token persistence pattern** — copy verbatim from yumyums `app.py` lines 47-71:
```python
def load_token():
    if TOKEN_FILE.exists():
        return json.loads(TOKEN_FILE.read_text())
    return None

def save_token(token):
    TOKEN_FILE.write_text(json.dumps(token))

def refresh_token(token):
    response = requests.post(
        TOKEN_URL,
        data={
            "grant_type": "refresh_token",
            "refresh_token": token["refresh_token"],
        },
        auth=(CLIENT_ID, CLIENT_SECRET),
        headers={"Content-Type": "application/x-www-form-urlencoded"},
    )
    response.raise_for_status()
    new_token = response.json()
    save_token(new_token)
    return new_token
```
Note: In yumyums `app.py` line 70, there is also `api_client.set_oauth2_token(new_token)` — omit that since the banking CLI does not maintain a global `api_client`. The `requests` import is aliased as `http_requests` in yumyums (`app.py` line 8) but should be plain `requests` here.
